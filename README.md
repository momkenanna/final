# ملف الإنجاز الإلكتروني - Vercel Template

قالب Next.js باللغة العربية RTL لملف إنجاز إلكتروني.

## التشغيل المحلي
```bash
npm install
npm run dev
```

## النشر على Vercel
1. ارفع المشروع إلى GitHub.
2. افتح Vercel > Add New Project.
3. اختر المستودع.
4. Framework Preset: Next.js.
5. اضغط Deploy.

## التعديل
- المحتوى الأساسي داخل: `app/page.tsx`
- الألوان والتصميم داخل: `app/globals.css`
- الصور والأدلة داخل: `public/evidence`
